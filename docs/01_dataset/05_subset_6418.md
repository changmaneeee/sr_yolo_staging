## 모든 실험에서 사용

다음 모든 실험에서 정확히 같은 6418장이 평가되었다:

| 실험 | 평가 set 지정 방식 |
|:--|:--|
| Fair eval (eval_arch.py) | `fixed_protocol.yaml`의 `dataset.filter: "labeled_only"` |
| Old Pipeline mAP (arch4_eval_ultralytics.py) | `--hr_data_yaml subset6418_hr_data.yaml --lr_data_yaml subset6418_lr_data.yaml` |
| Properbase mAP (arch4_eval_ultralytics.py) | 동일 |
| NMS sweep v1/v2 (arch4_eval_ultralytics.py) | 동일 |
| Sensitivity ablation | 동일 |
| Phase D/E interpolation/bonus eval | 동일 |

## fixed_protocol.yaml 의 명시

`configs/fixed_protocol.yaml` 발췌:
```yaml
dataset:
  lr_images: "/home/changmin/smart_airbus_data_lr/images/val"
  hr_images: "/home/changmin/smart_airbus_data/images/val"
  hr_labels: "/home/changmin/smart_airbus_data/labels/val"
  lr_labels: "/home/changmin/smart_airbus_data_lr/labels/val"
  filter: "labeled_only"
  expected_count: 6418
```

`expected_count: 6418`이 명시되어 있어, 평가 코드가 다른 개수를 마주칠 경우 즉시 감지 가능.

## Pre-flight Check

eval_arch.py에는 `pre_flight_check()` 함수가 있어 매 실행 시 다음을 확인:
